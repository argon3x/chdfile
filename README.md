# fcheck

***this script checks for duplicate files in a specific directory.***

### Note

- *Run script with python3*

        python3 fcheck.py -d [path directory with files]

### or

- *Give permission of execution*

        chmod u+x fcheck.py

- *Run script*

        ./fcheck.py -d [path directory with files]

- *for more help use*

        ./fcheck.py --help
